# 🏨 Hotel Booking Website (Spring Boot)

This is a hotel booking web application built using **Spring Boot**. The system allows users to register, log in, view available hotels and rooms, book a stay, and order food directly to their room. It uses **MySQL** as the backend database.

---

## 🚀 Features

- ✅ User authentication (sign up, log in, log out)
- 🏨 View list of hotels and hotel details
- 🛏️ View available rooms and amenities
- 🍽️ Order food to hotel rooms
- 📦 Book rooms online
- 💾 Persistent storage using MySQL

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot, Spring Security
- **Frontend:** Thymeleaf / React (depending on your setup)
- **Database:** MySQL
- **ORM:** Spring Data JPA / Hibernate
- **Authentication:** JWT / Spring Security
